isv-typesetting
===============

ISV Type-setting tools

This repository contains tools that can be used to typeset the text of the International Standard Version (ISV) Bible
(see ISV.org).

interest in or over the ISV text.  

Contact ISV.org for licensing details on the ISV text itself, and to obtain the ISV text that is necessary to actually
typeset the ISV bible.
